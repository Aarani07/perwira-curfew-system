Perwira Residential Curfew Detection System

Smart, Secure & Automated Curfew Monitoring for Student Residential Safety

Secure, Smart & Automated Curfew Monitoring — UTHM Perwira Residential College

About the System

The Perwira Residential Curfew Detection System is a secure, web-based application developed to automate the monitoring of student late check-ins at UTHM Perwira Residential College using facial recognition technology and role-based access control.

🎯 System Objective

To enhance curfew enforcement accuracy and reduce manual monitoring through an automated and reliable detection mechanism.

👥 Target Users

Students, Security Officers, and System Administrators within the UTHM Perwira Residential College.

🔐 Security Features

Secure authentication, role-based access control, encrypted data storage, and comprehensive audit logging.

Visi

Beriltizam menyediakan kemudahan yang mencukupi dan berkualiti selaras dengan keperluan universiti.

Misi

Kolej Kediaman Luar Kampus komited untuk menyemai budaya ilmu ke arah pembangunan mahasiswa syumul selaras visi dan misi universiti.

Piagam Pelanggan

  • Aduan kerosakan diselesaikan dalam tempoh 7 hari bekerja.
  • Pendaftaran kolej pelajar diselesaikan dalam masa kurang daripada 1 minit.
